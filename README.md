# sigscope

VCD (Value Change Dump) ファイルをターミナルで閲覧する TUI 波形ビューアです。
Bubble Tea を使った軽量 UI で、シグナル一覧・タイムライン・波形を表示します。ファイル変更を検知して自動で再読み込みします。

## 使い方

```bash
go run . path/to/file.vcd
```

またはビルドして実行:

```bash
go build -o sigscope .
./sigscope path/to/file.vcd
```

## 操作

- `q` / `Ctrl+C`: 終了
- `j` / `k` / `↑` / `↓`: シグナル移動
- `h` / `l` / `←` / `→`: 時間ウィンドウをスクロール
- `H` / `L` / `Shift+←` / `Shift+→`: ページ単位でスクロール
- `+` / `-` / `0`: ズームイン / ズームアウト / リセット
- `g` / `G`: 先頭 / 末尾へジャンプ
- `c`: カーソル表示の切替
- `[` / `]`: 前後の変化点へジャンプ
- `/`: 検索モード
- `s`: シグナル選択モード切替
- `space`: 表示/非表示の切替 (選択モードのみ)
- `a` / `A`: 全表示 / 全非表示 (選択モードのみ)
- `t`: 1行/2行表示の切替

## 仕様メモ

- VCD を読み込み、時間軸はシグナルの最終時刻に合わせます。
- VCD ファイルの変更を監視し、変更検出時に再パースします。
- 画面サイズ変更に追従します。

## 依存

- Go 1.24.3
- Bubble Tea / Lip Gloss / fsnotify
